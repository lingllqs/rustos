## LBA28 寄存器映射格式

LBA28(Logical Block Address 28-bit), 是 ATA/IDE 接口标准中用于寻址硬盘扇区的一种模式, 
使用**28**位二进制数来表示扇区地址,最大支持2^28个扇区

| 寄存器名称       | 主端口 (Primary) | 从端口 (Secondary) | 存放的 LBA 位段 | 说明                          |
|:-----------------|:----------------:|:------------------:|:---------------:|:------------------------------|
| Data             | 0x1F0            | 0x170              | -               | 读写数据用（不存地址）        |
| Error / Features | 0x1F1            | 0x171              | -               | 错误信息 / 功能设定           |
| Sector Count     | 0x1F2            | 0x172              | -               | 要读写的扇区数量 (0 表示 256) |
| LBA Low          | 0x1F3            | 0x173              | LBA [7:0]       | 地址的第 0-7 位               |
| LBA Mid          | 0x1F4            | 0x174              | LBA [15:8]      | 地址的第 8-15 位              |
| LBA High         | 0x1F5            | 0x175              | LBA [23:16]     | 地址的第 16-23 位             |
| Device / Head    | 0x1F6            | 0x176              | LBA [27:24]     | 地址的第 24-27 位 (高4位)     |
| Status / Command | 0x1F7            | 0x177              | -               | 读取状态 / 写入命令           |
